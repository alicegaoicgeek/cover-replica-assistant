# 提示词模板库

## 模板 A：深度复刻（默认）

最贴近参考图结构，适合对标成熟爆款封面。

```
参考上传的对标封面，本次为深度复刻：尽量保留原图的构图比例、标题位置、主体位置、画面密度、字体气质和视觉重心，但只复刻通用设计结构，不复制原图中的具体人物、签名、Logo、背景装饰、平台水印、原文案和独特识别元素。

请生成一张{品牌名}个人风格的{画幅}封面，画幅比例为{比例}。

标题文字为：
- {主标题}
- {副标题}
- {标签行}

构图说明：
{从Step2-3提取的构图描述}

字体说明：
{字体气质、字重、颜色、大小描述}

主体说明：
{主体位置、占比、风格描述}

氛围说明：
{色彩、光影、背景氛围描述}
```

## 模板 B：轻度借鉴

只参考色彩氛围和大致构图，标题和主体更自由。

```
参考上传的对标封面的色彩氛围和大致构图方向，但不严格复刻。仅借鉴其色彩系统、光影质感和视觉节奏。

请生成一张{品牌名}风格的{画幅}封面，画幅比例为{比例}。

标题文字：{主标题} / {副标题}

色彩方向：{从参考图提取的主色调}
构图方向：{大致布局，不严格复刻}
```

## 模板 C：风格迁移

保留自己品牌风格，仅迁移参考图的某个特定设计手法。

```
我已有固定的品牌封面风格。请参考对标封面中的{特定手法}（如：大字overlay的压迫感 / 上下夹击的构图 / 某一种色彩对比方式），将其迁移到我的品牌风格中。

品牌风格：{品牌现有风格描述}
要迁移的手法：{从参考图提取的特定手法}
标题文字：{标题}
画幅比例：{比例}
```
